PyEpoc by Morten A Steinsland <morten.a.steinsland@gmail.com>

PyEpoc is an attempt to make a Python wrapper for the Emotiv Epoc SDK headset. 

PyEpoc requires an Emotiv Epoc (SDK) headset and the SDK software.
